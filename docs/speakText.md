# Speak Text
## Notes
Speaks the given text.
## Syntax
```
speakText(text: <#String (Allows Variables)#>, wait: <#Boolean#>, rate: <#Double#>, pitch: <#Double#>, language: <#Type (WFSpeakTextLanguage)#>)
```
## Example
```
speakText(text: "Hello World!", wait: false, rate: 1, pitch: 1, language: en-US)
```
