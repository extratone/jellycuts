# Type Casting
## Notes
When using a variable you can add this onto the end of it to convert it too another type. All available types are case sensitive.
Available Types
1. AppStore
2. Article
3. Boolean
4. Contact
5. Date
6. Dictionary
7. Email
8. File
9. Image
10. iMedia
11. iProduct
12. Location
13. MapLink
14. Media
15. Number
16. PDF
17. PhoneNumber
18. Photo
19. Place
20. RichText
21. Webpage
22. Text
23. URL
24. VCard
## Syntax
```
.as(<#Casting Type#>)
```
## Example
```
var x = Shortcut Input.as(Text)
```
